# html-first-steps

This repository showcases a simple HTML form that collects basic user information, including :

  <li>Middle Name</li>
  <li>Last Name</li>
  <li>Gender</li>
  <li>Address</li>
  <li>Pincode</li>
  <li>Country</li>
  <li>Feedback</li>
